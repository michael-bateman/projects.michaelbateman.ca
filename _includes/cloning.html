Development Guide

If you want to clone the repository and work on the project, please follow these instructions exactly.

You will need the following to run this project:

  • Git
    • {% if project.ruby %}
  • Ruby
  • Jekyll
    • {% endif %} {% if project.python %}
  • Python
    • {% endif %}
    Most modern computers come with Git installed. If you do not have it installed, you can download it at https://git-scm.com/downloads.

    {% if project.ruby %} You will also need to download Ruby. Most modern Mac's come with Ruby pre-installed but if you do not have it, you can download it at https://www.ruby-lang.org/en/downloads/.

    Installing Jekyll is very simple. To do that, you need to open a command prompt and type the following.

    Note - to be able to install Jekyll you must have already installed Ruby. {% endif %} {% if project.python %} Make sure you have Python downloaded on your machine. Most modern Mac's come with Python pre-installed. You can check to see if Python is installed by typing:

    $ python

    If Python is installed, you should see the following message.

    Python 2.7.10 (default, Feb 8 2017, 20:04:21) [GCC 4.2.1 Compatible Apple LLVM 8.0.0 (clang-800.0.34)] on darwin Type "help", "copyright", "credits" or "license" for more information. >>>

    Press control + z to exit the Python interpreter. {% endif %}

    Note - Do not type the $ as it is a reminder that it is a command, to be entered on the command line.

    To clone the repository, type in the following:

    $ git clone https://github.com/{% if project.github_username %}{{ project.github_username }}{% else %}michael-bateman{% endif %}/{{ project.github }}

    {% if project.ruby %} Now, navigate to the directory by typing:

    $ cd {{ project.github }}

    Now, you need to serve Jekyll to start the server:

    $ jekyll serve

    Note: If you would like to serve the port on a different one than port 4000 type the following:

    $ jekyll serve --port <port_no> {% endif %} {% if project.python %} Now, cd into the repository by typing:

    $ cd {{ project.github }}

    Make sure you have the correct modules installed, and type the following when ready to run the program.

    $ python <file_name> {% endif %}
